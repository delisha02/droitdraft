# DroitDraft

DroitDraft is an AI-powered legal document generation platform.

## Monorepo Structure

This repository is a monorepo containing the following components:

-   `frontend/`: The Next.js web application.
-   `backend/`: The Python backend, powered by FastAPI.
-   `shared/`: Shared configurations and utilities between the frontend and backend.
-   `docs/`: Project documentation.
-   `prompts/`: Prompts for the AI models.
